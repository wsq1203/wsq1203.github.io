JumpServer安装

官方说明

安装要求

  • 硬件配置：2CPU，4G内存，50G硬盘（最低）
  • 操作系统：Linux发行版x86_64
  • Python = 3.6x
  • MySQL Server >= 5.6 或者 Mariadb Server >= 5.5.56 数据库编码要求 utf8，新版要求 5.7 以上
  • Redis：新版要求 6.0 以上

安装方法

官方提供了多种安装方法

  • 手动部署：按组件逐个实现
  • 急速部署：资产数量不多，或者测试体验的用户请使用脚本快速部署
  • 容器部署：基于docker实现
  • 分布式部署：使用大型环境
